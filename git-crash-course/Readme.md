## Cloning

There are 3 ways to clone: HTTPS, SSH and Github CLI

Since we're using Github Codespaces we'll create a temporary directory in our workspace
```sh
mkdir /workspace/tmp
cd /workspace/tmp
```

### HTTPS
```sh
git clone https://github.com/DonutsInTheSnow/Github-examples.git
```
